# Images - IVOIRE.AI

## Logo

Placez le fichier logo dans ce dossier avec le nom : `logo.png`

### Spécifications recommandées :
- Format : PNG avec transparence
- Taille : 200x200px minimum (pour une bonne qualité)
- Nom du fichier : `logo.png`

### Utilisation :
Le logo est utilisé dans :
- La barre de navigation (navbar)
- Le footer
- Les meta tags Open Graph et Twitter Cards

### Alternative :
Si vous n'avez pas encore le logo, vous pouvez :
1. Utiliser un service en ligne pour créer un logo temporaire
2. Utiliser un placeholder jusqu'à ce que le logo final soit disponible
3. Le logo s'affichera automatiquement une fois le fichier `logo.png` placé dans ce dossier
